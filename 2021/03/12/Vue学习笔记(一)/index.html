Vue详解

Vue概述

  • Vue是一套用于构建用户界面的渐进式JavaScript框架(渐进性：逐步实现新特性)
  • Vue被设计为可以自底向上逐层应用
  • Vue的核心库只关注视图层(HTML+CSS+JS)，方便与第三方库或既有项目整合
  • MVVM+虚拟Dom
  • 网络通信：axios
  • 页面跳转：vue-router
  • 项目管理：vuex

CSS预处理器

  • 用一种专门的编程语言，进行Web页面样式设计，再通过编译器转化为正常的CSS文件，以供项目使用
  • 常见：LESS(基于Node.js)

前端MVVM

  • M：模型
  • V：视图
  • VM(C)：视图模型，前后双向绑定

UI框架

  • Ant-Design基于React的UI框架
  • Element、iview、ice
  • Bootstrap
  • AmazeUI

JavaScript

  • Babel：JS编译工具
  • WebPack：模块打包器

Node.js

  • Express：Node.js框架
  • NPM：项目综合管理工具，类似于Maven
  • YARN:NPM的替代方案

第一个Vue程序

注意：使用IDEA开发

  • 新建一个空项目，在setting的插件下载安装vue.js

  • 导入vue.js

  • 新建一个div标签

  • 新建一个vm对象

  • 把数据从(使用双括号)div标签中取出(数据双向绑定)

为什么要使用MVVM：

  • 低耦合：View可以独立于Model变化和修改，一个ViewModel可以绑定到不同的View上，当View变化的时候Model可以不变，当Model变化的时候View也可以不变
  • 可复用：可以把一个视图逻辑放到一个ViewModel里面，让很多View复用
  • 独立开发：专注于ViewModel和页面的分开开发
  • 可测试
